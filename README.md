# CUT (ID: 13, TYPE: C)

## Characteristics

### Validation
- ✔️ Use one, and only one of -b, -c or -f.
- ✔️ Invalid option
- ✔️ Verify first word "cut"
- ✔️ Verify that ranges are placed after options
- ✔️ -b, -c or -f commands must not appear 2 times
- ✔️ Verify if the files exist
